VSCode 플러그인 리스트
Atom Keymap : 아톰의 키 설정을 불러오는 플러그인
Vetur : Vue.js 플러그인
Night Owl : 코드 하이라이팅 플러그인(코드 자체 색상이 변경됨)
Material Dark Syntax : 코드 하이라이팅 플러그인
Material Icon Theme : 폴더 아이콘 테마 (Google Material Icon Theme - js, vue, html등등 vscode 내에서 아이콘 표시)
ESLint : 자바스크립트 문법 검사 플러그인
TSLint : 타입스크립트 문법 검사 플러그인
Auto Close tag : HTML 태그 자동 닫기 플러그인 //시작태그를 입력하면 끝내그를 입력해줌
Live Server : 정적 파일을 로컬 서버에 올리고 자동 갱신해주는 플러그인
